Appearance

Signal Processing

V1 uses a 5-stage pipeline (Kalman → Median → MA → EMA → Step limiter). V2 uses a single 2nd-order Bessel IIR (RotorHazard method) with midpoint lap timestamp. Changes take effect immediately without reboot.

Detection

V1 debounce parameters. Not used in V2 mode.

Integration

Firmware Update

Installed version v —

FPVRaceOne requires two files to be uploaded when updating: firmware.bin (application) and littlefs.bin (web interface & assets). Both are available on the GitHub releases page.

📱 Phone / Tablet
  1. Click GitHub Releases below — your phone's cellular connection may not be active while connected to FPVRaceOne WiFi and you should disconnect first.
  2. Download firmware.bin and littlefs.bin to your phone
  3. Click Open ElegantOTA below
  4. Upload firmware.bin — select Firmware mode
  5. Upload littlefs.bin — select Filesystem mode
💻 Laptop / Desktop
  1. Disconnect from FPVRaceOne WiFi
  2. Click GitHub Releases below and download both .bin files
  3. Reconnect to FPVRaceOne WiFi
  4. Click Open ElegantOTA below
  5. Upload firmware.bin — select Firmware mode
  6. Upload littlefs.bin — select Filesystem mode
